Sample update request

Whenever Kactus checks for a new update, it will make a request to the update server of the following form: 
GET https://kactus-autoupdater.herokuapp.com/update/darwin_macos/0.1.0 HTTP/1.1
Host: kactus-autoupdater.herokuapp.com
Accept: application/json
Accept-Language: en-us
Connection: keep-alive
Accept-Encoding: gzip, deflate
User-Agent: Kactus/0.1.0 CFNetwork/811.4.18 Darwin/16.5.0 (x86_64)
